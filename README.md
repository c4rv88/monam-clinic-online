# Monamê Clinic

Site institucional da Monamê Clinic — 25 anos de tecnologia e cuidado personalizado em estética avançada.

## Stack

- Vite + React + TypeScript
- Tailwind CSS + shadcn/ui
- React Router

## Desenvolvimento

```sh
bun install
bun run dev
```

## Build

```sh
bun run build
```
